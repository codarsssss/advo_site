Защита адвоката по уголовным делам

  • Участие адвоката в уголовных делах обеспечивает справедливость в расследовании дел правоохранительными органами и рассмотрении дел судами, а также защиту прав и интересов обвиняемых от незаконного уголовного преследования либо наказания. Важно позаботиться о наличии адвоката уже в первые дни следствия, чтобы избежать серьезных юридических рисков!
  • МКА "Партнер Групп" предоставляет профессиональную защиту адвокатами по уголовным делам на стадиях доследственной проверки, предварительного следствия и суда
    • Адвокат по делам о мошенничестве (Адвокат по статье 159 УК РФ)
    • Адвокат по делу о присвоении и растрате (Адвокат по статье 160 УК РФ)
    • Адвокат по делам о наркотиках (Адвокат по статьям 228 УК РФ , 228.1 УК РФ , 229 УК РФ о контрабанде наркотиков)
    • Адвокат по делам о взятке (Адвокат по статьям 290, 291 УК РФ)
    • Адвокат по делам о коммерческом подкупе (Адвокат по статьям 204, 204.1 УК РФ)
    • Адвокат по делам об убийстве по статье 105 УК РФ (в том числе участие в суде присяжных)
    • Адвокат по делу о тяжком вреде здоровью (Адвокат по статье 111 УК РФ)
    • Адвокат по делам об оказании услуг и продаже товаров, не отвечающих требованиям безопасности (Адвокат по статье 238 УК РФ)
    • Адвокат по делам о вымогательстве (Адвокат по статье 163 УК РФ)
    • Адвокат по делам о незаконной банковской деятельности (Адвокат по статье 172 УК РФ)
    • Адвокат делам об уклонении от уплаты налогов (Адвокат по статьям 198, 199 УК РФ)
    • Адвокат по делам о хулиганстве (Адвокат по статье 213 УК РФ)
    • Адвокат по делам об экстремизме (Адвокат по статьям 282, 282.1, 282.2 УК РФ)
    • Адвокат для по военным делам (Военный адвокат для призывников и военнослужащих по статьям 337, 338, 339 УК РФ)
  • Стоимость СРОЧНОГО выезда адвоката к месту следственных действий от 30 000 рублей. Иные расценки определяются после предварительной беседы.
